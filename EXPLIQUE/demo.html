💬 Chat en Temps Réel - Démonstration

Application complète de discussion développée avec HTML, CSS, JavaScript et PHP

🚀 Comment tester l'application

  1. Ouvrez l'application dans votre navigateur
  2. Entrez un nom d'utilisateur (ex: "Alice")
  3. Cliquez sur "Rejoindre" pour vous connecter
  4. Tapez un message et appuyez sur Entrée
  5. Ouvrez un second onglet pour simuler un autre utilisateur
  6. Testez la conversation en temps réel !
